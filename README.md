\# Email Validator Pro



Advanced email validation tool for Office 365 with comprehensive validation features.



\## Features



\- ✅ RFC 5322 syntax validation

\- ✅ DNS/MX record verification

\- ✅ Disposable email detection

\- ✅ Role-based email flagging

\- ✅ Bulk validation support

\- ✅ Export to CSV/TXT

\- ✅ Real-time search and filtering

\- ✅ Scoring algorithm (0-100)



\## Installation

```bash

npm install

```



\## Development

```bash

npm run dev

```



\## Build

```bash

npm run build

```



\## Deploy to Vercel



1\. Push to GitHub

2\. Import repository in Vercel

3\. Deploy automatically
